# 🏗️ Construction Planning System

## AI-Powered Construction Project Planning with IBM Granite 3.3 2B

### 📌 Overview

Construction Planning System is an intelligent AI-powered platform designed to simplify and optimize construction project planning. The system leverages **IBM Granite 3.3 2B** running locally through **Ollama** to generate accurate project estimates, resource allocation plans, cost analysis, material requirements, construction schedules, and architectural blueprints.

The platform helps contractors, builders, project managers, and developers make informed decisions by providing comprehensive planning recommendations based on project specifications such as built-up area, number of floors, and project timelines.

## ⚙️ Installation

### 1. Clone Repository

```bash
git clone https://github.com/yourusername/construction-planning-system.git
cd construction-planning-system
```

### 2. Create Virtual Environment

```bash
python -m venv venv
```

Activate Environment:

Windows

```bash
venv\Scripts\activate
```

Linux/Mac

```bash
source venv/bin/activate
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Install Ollama

Download and install Ollama from:

https://ollama.ai

### 5. Pull IBM Granite Model

```bash
ollama pull granite:3.3-2b
```

### 6. Run Application

```bash
python app.py
```

Open:

```bash
http://localhost:5000
```
